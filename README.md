## Features

* User system
  1. Sign-up
  2. Login
* Movie catalog
* Movie details
* Movie reviews
* Movie recommendations
* Actor catalog
* Actor details
* Load scripts
  1. Genres
  2. Movies
* Dockerized project
* Deployment
  1. Deployed on a container on EC2 instance, accessable through [custom dns](gutelekture.ddns.net:8000/) (when EC2 instance is active)
  2. Using RDS database (Postgres)
